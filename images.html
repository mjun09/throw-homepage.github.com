Images

This is a new project. Please let us know if something is not clear so we can fix it. Your comments will help us greatly!
For images you use the img element. This element has an attribute named src where you specify the URL of the image. Here is an example: 
<img src="http://throw-platform.github.com/img/turkey.png" />
To add gravity we need to grav tag. 
<img class="grav" src="http://throw-platform.github.com/img/turkey.png" />
