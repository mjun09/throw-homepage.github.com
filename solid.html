Solid Tag

This is a new project. Please let us know if something is not clear so we can fix it. Your comments will help us greatly!

We provided another unique identifier, solid, to help you get started. Solid doesn't provide gravity but rather allows a div with a gravity identifier to interact. Only divs with gravity (or solid) can interact with each other. A grav identified div will not interact with div that does not share either of these properties.

The solid tag has many uses. It can serve as a wall, platform etc to help add more fun to your project. It is added the same way as the grav tag by adding class=. Keep in mind, you can string together multiple classes within class= separated with a space.

To show how important and useful solid can be, remove the solid identifier on the right hand side of the screen.

< div style="background: red; width: 50%; height: 50px; position: absolute; bottom: 10%; right: 0"></div>

As you can see the gravity div simply travels through the red div rather than bounces off of it.
